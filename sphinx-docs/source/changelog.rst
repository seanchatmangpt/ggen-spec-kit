Changelog
=========

All notable changes to Specify CLI are documented here.

The format is based on `Keep a Changelog <https://keepachangelog.com/en/1.0.0/>`_,
and this project adheres to `Semantic Versioning <https://semver.org/spec/v2.0.0.html>`_.

[0.0.25] - 2025-12-25
---------------------

Added
~~~~~
- Comprehensive Sphinx documentation with autodoc
- DSPy LaTeX integration with 10-agent maximum concurrency
- Hyperdimensional computing metrics and embeddings
- JTBD framework integration
- Process mining with pm4py
- Three-tier architecture (commands/ops/runtime)
- Full OpenTelemetry instrumentation
- RDF-first development with ggen v5.0.2

Changed
~~~~~~~
- Refactored commands to three-tier architecture
- Enhanced observability with detailed spans
- Improved error handling and recovery

Fixed
~~~~~
- Type hints coverage
- Test coverage improvements
- Documentation completeness
