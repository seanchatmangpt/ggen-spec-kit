## Pattern Languages

### Christopher Alexander

**A Pattern Language: Towns, Buildings, Construction** (1977)
: Christopher Alexander, Sara Ishikawa, Murray Silverstein
: The foundational work on pattern languages. 253 patterns for architecture.

**The Timeless Way of Building** (1979)
: Christopher Alexander
: The philosophical companion to A Pattern Language. Introduces "the quality without a name."

**The Nature of Order** (2002-2005)
: Christopher Alexander
: Four-volume work on the nature of living structure.

### Software Patterns

**Design Patterns: Elements of Reusable Object-Oriented Software** (1994)
: Erich Gamma, Richard Helm, Ralph Johnson, John Vlissides (Gang of Four)
: The influential software patterns book. Different style from Alexander—more structural.

**Pattern-Oriented Software Architecture** (1996-2007)
: Frank Buschmann et al.
: Five-volume series on architectural patterns.

## Jobs To Be Done

**Competing Against Luck** (2016)
: Clayton Christensen, Taddy Hall, Karen Dillon, David Duncan
: The definitive JTBD book from the framework's creator.

**The Jobs To Be Done Playbook** (2019)
: Jim Kalbach
: Practical techniques for applying JTBD.

**Demand-Side Sales 101** (2020)
: Bob Moesta
: JTBD applied to understanding customer demand.

## Semantic Web & RDF

**RDF 1.1 Primer** (W3C)
: [https://www.w3.org/TR/rdf11-primer/](https://www.w3.org/TR/rdf11-primer/)
: Introduction to RDF concepts.

**SHACL Specification** (W3C)
: [https://www.w3.org/TR/shacl/](https://www.w3.org/TR/shacl/)
: Shapes Constraint Language specification.

**SPARQL 1.1 Query Language** (W3C)
: [https://www.w3.org/TR/sparql11-query/](https://www.w3.org/TR/sparql11-query/)
: Query language for RDF.

**Turtle Syntax** (W3C)
: [https://www.w3.org/TR/turtle/](https://www.w3.org/TR/turtle/)
: Human-readable RDF serialization.
