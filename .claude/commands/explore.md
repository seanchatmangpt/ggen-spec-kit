# Explore Codebase

Explore and understand the codebase structure and patterns.

## Usage
```
/explore [TOPIC]
```

## Arguments
- `$ARGUMENTS` - What to explore (e.g., "authentication", "database", "API endpoints")

## Instructions

Explore the codebase to understand: $ARGUMENTS

Use the Explore agent type for thorough investigation:
1. Search for relevant files using Glob and Grep
2. Read key files to understand patterns
3. Map dependencies and relationships
4. Identify entry points and flows

Provide a comprehensive report including:
- Relevant files and their purposes
- Key classes/functions involved
- Data flow and architecture
- Code patterns used
- Recommendations or observations

If no topic provided, give an overview of the entire codebase structure.
